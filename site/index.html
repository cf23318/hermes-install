Hermes Agent 内部安装

这个安装器用于 macOS 和 Windows 原生设备（非 WSL2）。它会自动检测代理、安装 Hermes Agent、 配置 DeepSeek 官方 endpoint，并引导用户选择聊天工具。聊天工具推荐使用飞书， 也可以选择 Hermes 支持的其他平台。

下载 macOS 版本 下载 Windows 版本

Quick Start

  1. 先打开你的翻墙/代理软件，确保能访问 Google。
  2. 访问 https://www.google.com/，确认代理已经生效。
  3. 按自己的系统点击 下载 macOS 版本 或 下载 Windows 版本。
  4. macOS 按照下面的 mac 安装指南 操作；Windows 右击 HermesAgentInstaller.ps1 文件，选择“使用 PowerShell 运行”。
  5. 选择 1) 安装/修复 Hermes，并配置 DeepSeek 与聊天工具（推荐飞书）。
  6. 输入自己的 DeepSeek API Key。
  7. 按向导选择聊天工具。推荐选择 10. Feishu / Lark。
  8. 如果聊天工具没有响应，重新运行安装器并选择 4) 检查 Hermes/Gateway 状态与错误日志。
如果 Windows 执行策略拦截，请打开 PowerShell，先运行 Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass，再运行 .\HermesAgentInstaller.ps1。
